# Legal docs (templates)

These documents are **templates** generated for the app **“Polls That Shouldn’t Exist”**. They are **not legal advice**.

Before publishing, fill in the placeholders:

- `[OPERATOR_LEGAL_NAME]` (your personal/legal entity name)
- `[OPERATOR_ADDRESS]`
- `[CONTACT_EMAIL]`
- `[WEBSITE_URL]` (your production domain)
- `[JURISDICTION]` (e.g. “Delaware, USA” / “England and Wales”)
- `[EFFECTIVE_DATE]`

Optional (if applicable):
- `[DPO_OR_PRIVACY_CONTACT]`
- `[EU_REPRESENTATIVE]`

Files:
- `legal/terms-of-service.md`
- `legal/privacy-policy.md`
